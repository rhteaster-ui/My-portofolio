Tentang Saya

Saya adalah seorang pengembang web mandiri (self‑taught) yang berfokus pada pembuatan web app, PWA, dan website statis interaktif dengan pendekatan efisien, ringan, dan fungsional, terutama untuk perangkat dengan spesifikasi rendah.

Saya memulai perjalanan di dunia teknologi secara otodidak, tanpa mentor langsung dan dengan keterbatasan perangkat. Dalam proses tersebut, saya mengembangkan kemampuan untuk mendesain, membangun, mengoptimalkan, dan mendeploy aplikasi web menggunakan bantuan AI sebagai productivity tool, bukan sekadar generator kode.

Fokus utama saya bukan hanya “membuat website terlihat keren”, tetapi memastikan web benar‑benar berguna, mudah dipahami, dan stabil digunakan oleh pengguna umum. Semua itu dilakukan full device only.
